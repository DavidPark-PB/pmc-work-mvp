1. 시스템 소개

PMC-AUTO는 한국 도매 플랫폼에서 소싱한 상품을 해외 쇼핑몰(eBay, Shopify 등)에 자동으로 리스팅하는 시스템입니다.

Staff가 하는 일:

  • 관리자가 할당한 상품을 확인하고 관리
  • 상품 정보를 수정하고 해외 플랫폼에 업로드

Staff가 접근할 수 있는 메뉴:

  • 상품 관리, CSV 업로드, 히스토리, 휴지통
설정, 토큰 관리, 직원 관리, 작업 로그는 관리자 전용 메뉴입니다. 사이드바에 "관리" 섹션은 표시되지 않습니다.

2. 로그인

URL: /login

  1. 관리자가 생성해준 아이디와 비밀번호를 입력합니다.
  2. 로그인 버튼을 클릭합니다.
  3. 첫 로그인 시 이름 설정 모달이 표시됩니다. 본인 이름을 입력해주세요.
  4. 로그인 성공 시 자동으로 상품 관리 화면으로 이동합니다.
비밀번호를 잊은 경우, 관리자에게 초기화를 요청하세요.

3. 상품 관리

URL: /

상품 관리는 Staff가 가장 많이 사용하는 화면입니다. 할당받은 상품을 확인하고, 수정하고, 플랫폼에 업로드합니다.

3-1. 탭

  1. 전체 — 모든 상품을 표시합니다.
  2. 업로드 대기 — 관리자가 할당해준 상품 중 아직 업로드하지 않은 목록입니다.
  3. 업로드 완료 — 플랫폼에 업로드가 완료된 상품입니다.
  4. 판매 취소 — 판매가 취소된 상품입니다.
업로드 작업은 주로 업로드 대기 탭에서 진행합니다.

3-2. 상단 툴바

  1. KR/EN 토글 — 상품명을 한글과 영문으로 전환합니다.
  2. 일괄 번역 — 선택한 상품의 한글 상품명을 영문으로 자동 번역합니다 (Gemini AI 사용).
  3. 검색 — 상품명으로 검색합니다.

3-3. 상품 테이블

  1. 체크박스 — 나에게 할당된 상품만 선택할 수 있습니다.
  2. 인라인 편집 — 상품명, 원가를 직접 클릭하면 수정할 수 있습니다.
  3. 상태 — 상품의 현재 업로드 상태를 확인합니다.
미할당 상품이나 다른 직원에게 할당된 상품은 체크박스가 잠겨 있습니다. 이는 정상 동작입니다.

3-4. 액션바 (업로드)

  1. 업로드할 상품을 체크박스로 선택합니다.
  2. 액션바에서 플랫폼을 선택합니다 (eBay, Shopify, Alibaba, Shopee 중 하나 이상).
  3. 업로드 버튼을 클릭하면 업로드가 시작됩니다.
  4. 업로드 진행 화면에서 실시간 진행률을 확인할 수 있습니다.
상품을 선택해야 액션바가 활성화됩니다.

3-5. Description 미리보기

업로드 대기 탭에서 상품명 옆의 메모장 아이콘(📋)을 클릭하면 해당 상품의 Description을 미리볼 수 있습니다.

  • 아이콘 클릭 시 AI가 자동으로 Description을 생성합니다.
  • 생성된 Description은 개별 상품 설명 + 공통 정책 템플릿이 합쳐진 최종 형태입니다.
  • 상단 드롭다운으로 플랫폼별 결과(eBay, Shopify, Shopee 등)를 전환하여 확인할 수 있습니다.
  • Shopee는 HTML을 지원하지 않으므로 플레인 텍스트로 자동 변환되어 표시됩니다.
Description은 업로드 시 자동으로 생성되므로 별도 작업 없이 바로 업로드할 수 있습니다. 미리보기는 확인용입니다.

3-6. Description 일괄 생성 / 일괄 번역

상단 툴바의 버튼으로 일괄 작업을 실행할 수 있습니다:

  • 일괄 번역: 미번역 상품의 한글 제목을 영문으로 번역합니다. 번역 시 Description도 함께 생성됩니다.
  • Desc 생성: Description이 없는 기존 상품에 AI로 Description을 일괄 생성합니다.
두 버튼 모두 진행 모달이 표시되며, 실시간으로 진행률을 확인할 수 있습니다.

4. CSV 업로드

URL: /upload-csv

CSV 파일로 상품 데이터를 일괄 등록할 수 있습니다.

업로드 방법

  1. CSV 파일을 드래그앤드롭 하거나, 영역을 클릭하여 파일을 선택합니다.
  2. 업로드 후 데이터 미리보기를 확인합니다.
  3. 확인 후 DB 등록을 진행합니다.
  4. 등록이 완료되면 상품 관리의 업로드 대기 탭에 표시됩니다.

5. 업로드 결과

플랫폼 업로드가 완료되면 결과 화면이 표시됩니다.

결과 요약

  1. 전체 — 업로드를 시도한 총 상품 수
  2. 성공 — 정상적으로 업로드된 상품 수
  3. 실패 — 업로드에 실패한 상품 수
실패한 상품이 있다면 상세 테이블에서 실패 사유를 확인하고, 관리자에게 문의하세요.

6. 히스토리

URL: /history

  1. 그동안 작업한 업로드 잡 목록을 확인할 수 있습니다.
  2. CSV 업로드 이력도 확인할 수 있습니다.
  3. 각 잡을 클릭하면 상세 결과를 볼 수 있습니다.

7. 휴지통

URL: /trash

  1. 삭제한 상품을 복원할 수 있습니다.
  2. 복원하지 않을 상품은 완전 삭제할 수 있습니다.
완전 삭제한 상품은 복구할 수 없습니다.

8. 자주 묻는 질문 (FAQ)

Q. 내 상품이 안 보여요

관리자가 아직 상품을 할당하지 않았을 수 있습니다. 관리자에게 상품 할당을 요청하세요.

Q. 업로드 실패했어요

히스토리 페이지에서 해당 업로드 잡의 결과를 확인하세요. 실패 사유를 확인한 뒤 관리자에게 문의해주세요.

Q. 비밀번호를 잊었어요

Staff가 직접 비밀번호를 변경할 수는 없습니다. 관리자에게 비밀번호 초기화를 요청하세요.

Q. 다른 사람 상품이 잠겨있어요

정상 동작입니다. 본인에게 할당된 상품만 선택하고 수정할 수 있습니다. 해당 상품이 필요하다면 관리자에게 재할당을 요청하세요.
