🖼️ 쿠팡 상세페이지 이미지 추출기

쿠팡 상품 페이지에서 썸네일 + 상세페이지 이미지 URL을 한 번에 추출합니다.

1북마클릿 등록 (한 번만)

아래 버튼을 북마크 바로 드래그하세요:

📷 쿠팡 이미지 추출

2사용 방법

  1. 쿠팡 상품 페이지로 이동
  2. 상세페이지를 아래로 스크롤하여 이미지가 로드되게 함
  3. 북마크 바에서 📷 쿠팡 이미지 추출 클릭
  4. 오른쪽 상단에 이미지 URL 목록 표시 → 복사 클릭
  5. PMC 대시보드 > 상세페이지 재구성 > 이미지 URL 입력란에 붙여넣기
  6. Shopify CDN 업로드 클릭

A대안: 크롬 콘솔에서 직접 실행

북마클릿이 안 되면, 쿠팡 상품페이지에서 F12 → Console 탭에 붙여넣기:

var imgs=new Set;document.querySelectorAll('img[src*=coupangcdn],.product-detail-content-inside img').forEach(e=>{var s=e.src||e.dataset.src||e.getAttribute('data-img-src')||'';if(s.match(/^https?:/)){s=s.replace(/\/q\/\d+/,'/q/90');imgs.add(s)}});var list=[...imgs];copy(list.join('\n'));console.log(list.length+'장 복사됨!',list);

실행하면 자동으로 클립보드에 복사됩니다.
